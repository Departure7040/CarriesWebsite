# Google Business Profile — Example Posts

**Purpose:** 12 ready-to-adapt GBP post drafts, mixing content types per `/reports/02_local_seo_audit.md` Section (b)(12). Target cadence: 2–4 posts/month minimum, ideally weekly. `[best-practice]`

**Pending:** These posts cannot go live until the GBP exists (`/reports/02_local_seo_audit.md` Section a-1). Draft now so publishing can start immediately once the profile is created and verified.

**UTM pattern for every post's link:**
```
https://carriebilleaud.exprealty.com/[page]?utm_source=google&utm_medium=organic&utm_campaign=gbp_post_[topic]_[yyyy_mm]
```
Example: `?utm_source=google&utm_medium=organic&utm_campaign=gbp_post_marketupdate_2026_07`

All statistics below are placeholders marked `[client-confirm]` — do not publish with real numbers until verified per `/data/known_claims.yaml`.

## Posting Cadence Notes

`[best-practice]` Rotate through content types rather than repeating the same format back-to-back — aim for roughly: 1 market update + 1 neighborhood spotlight + 1 tip post + 1 listing/sold or seasonal post per month, at minimum. GBP posts expire from the main profile view after 7 days (they remain in a post history tab), so weekly posting keeps the profile looking active to anyone landing on it directly, which is itself a freshness/prominence signal.
